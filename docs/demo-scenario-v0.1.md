## 2. ライブデモ 3本（仕込みなし推奨）

### 🏆 デモ1: LINE Bot（既存社内プロダクト構造完全流用）

**実演フロー**：
1. 審査員に QR コード見せて「友達追加」してもらう
2. 審査員に「今気になるニュース記事のURL」を貼ってもらう（仕込みなし）
3. DeepFact Validator が 3秒以内に返信：
   - 🔍信頼度スコア（高/中/低/警告）
   - 📝要約
   - 🔎構造観察3点
   - 📚一次情報URL
   - 🛡️免責文（「人ではなく構造を見る」）

**狙い**：既存社内プロダクト実績（社長が運用中）＝Cloud Run min-instances=1 で 30秒前後の体感速度＝実機の信頼性

### 🥈 デモ2: Chrome Extension（突き抜けた体験価値）

**実演フロー**：
1. 審査員にニュースサイト開いてもらう（例：Yahoo!ニュース・任意）
2. 拡張機能の「このページを解析」ボタンクリック
3. ポップアップに：
   - 信頼度バー（赤/黄/緑）
   - 構造観察リスト
   - 一次情報リンク
   - 利害関係マップ（Phase 2では関係性グラフ）

**狙い**：「ユーザーが疑問を持つ前に介入」＝チャットボットとの根本的差別化

### 🥉 デモ3: Postmortem 機能（DevOpsフレーム🅰）

**実演フロー**：
1. 「あなたが直近24時間で見た記事」の振り返り
2. SRE風 Postmortem 形式で出力：
   - 警告履歴（severity: INFO/WARN/CRITICAL）
   - 構造的傾向（「信頼度低位の記事を 3件見ました」）
   - 改善提案（「次は一次情報URLを確認することを推奨」）

**狙い**：DevOpsテーマ直撃＝「情報のObservability」フレーム実装

## 4. 法的リスク回避（Brand Soul① 品格 継承）

### ❌ 言わない
- 「このジャーナリストは○○から金もらってる」
- 「この記事は嘘」
- 発信者個人を断定的に指摘

### ✅ 言う
- 「この記事の出所と過去発言には XX という構造的関係がある」
- 「一次情報と比較すると○○の点で論調が異なる構造」

## 8. 🆕 v0.2 実機検証データ＋朝の社長用テストパターン（2026-06-19 23:35 追記）

### 実機検証エビデンス（コンテスト提出時に提示）

| パターン | 判定 | 総合スコア | 出所信頼度 | 中立性 | 事実整合性 |
|---------|------|-----------|-----------|--------|-----------|
| フェイク主張（ジェノサイド断定） | **警告** | **20%** | 0.50 | 0.00 | 0.00 |
| 厚労省一次情報URL | **高** | **96%** | 0.95 | 0.95 | 0.98 |
| 中立報道風（両論併記） | **中** | **50%** | 0.50 | 0.50 | 0.50 |
| 誇張広告（がんサプリ） | **警告** | **20%** | 0.50 | 0.00 | 0.00 |

→ **76ポイント差**で完全に入力依存判定。証拠 raw JSON: `test-results/0[1-4]-*-result.json`

### 🌅 朝の社長用 LINE Bot 実機テスト 5パターン（コピペで送れる）

LINE Bot「DeepFact Validator」のトークに以下を1つずつ送って、応答スクショを取得→ピッチ素材化。

#### パターン A: 政府公式（高信頼度期待）
```
https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/iryouhoken/iryouhoken01/index.html
```
→ 信頼度 **高 90%以上**

#### パターン B: 中立報道風テキスト（中信頼度期待）
```
政府は2026年6月、医療制度改革に関する新たな指針を発表した。指針には診療報酬体系の見直しが含まれており、関係団体からは賛否両論の声が上がっている。厚生労働省によると2027年度から段階的に施行される。
```
→ 信頼度 **中 50%前後**

#### パターン C: 煽動的フェイク（警告期待）
```
ワクチンの真実が暴露された！厚労省が隠していた重大な副作用、医師たちが沈黙を破る。これを知ったら誰もワクチンを打たない。今すぐ拡散してください！
```
→ 信頼度 **警告 30%以下**

#### パターン D: 商業誇張広告（警告期待）
```
たった3日でガンが消えた！医師も驚愕の奇跡のサプリ、今だけ50%OFF！購入はこちら。副作用なし、すべての癌に効きます。
```
→ 信頼度 **警告 25%以下**

#### パターン E: 学術的論調（高〜中期待）
```
最新の研究によれば、糖質制限ダイエットは短期的な体重減少効果が認められる一方で、長期的な心血管系への影響については複数の系統的レビューで結論が分かれている。American Heart Association は適度な糖質摂取を推奨している。
```
→ 信頼度 **中〜高 60-75%前後**

### デモ時の語り口例（コンテスト本番）

> 「DeepFact Validator は **76ポイントの判定差** を機械的に出します。フェイク主張は **20%**、厚労省一次情報は **96%**。同じ Bot が、同じ仕組みで、入力ごとにこれだけ差をつけます。しかも判定の根拠は『論理的飛躍』『煽動的構造』『権威の曖昧な利用』のような**構造観察**として可視化される。発信者個人を断定せず、構造を可視化する。これが情報の Observability です。」

### スコア解釈ガイド

| スコア帯 | ラベル | 意味 |
|---------|--------|------|
| 0.75-1.00 | 高 | 一次情報・公的機関・両論併記が揃った構造 |
| 0.50-0.74 | 中 | 一定の検証可能性あり・断定的でない構造 |
| 0.30-0.49 | 低 | 出典不明・断定過多・要慎重評価の構造 |
| 0.00-0.29 | 警告 | 検証不能の主張多数・煽動的・無条件受容を促す構造 |
